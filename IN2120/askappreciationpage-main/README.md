## Ikke komplett guide for å kjøre Docker image

1. Nettsiden bruker Node: [install npm](https://nodejs.org/en/download/package-manager)
2. Du trenger denne for å kjøre skriptet jeg lagde som setter opp container: npm install shelljs
3. Start docker
4. (Om du har mac og docker ikke funker:
   - brew install Colima
   - Colima start
   - Evt. podman
6. Dette er for å kjøre nettsiden: node load_and_run_app.js
